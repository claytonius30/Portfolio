Finance Tracker Frontend

The final project for my .NET Applications class involved developing a client-side UI for my Finance Tracker mobile app using .NET MAUI (Multi-platform App User Interface). I followed and adapted material from the PluralSight course "Building .NET MAUI Applications with MVVM" by Lindsey Broos to build the app. Fully understanding the Model View ViewModel (MVVM) pattern and implementing authentication took time, but in the end, I smoothly integrated with the backend API (see below). Despite the framework's code-intensive nature, the strong separation of concerns and single code base ensured deployability across platforms. Learning this framework has equipped me with invaluable skills for future app development.
